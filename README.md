## 🚀 Project Overview

**IdeaValid (IdeaVault)** is a modern SaaS-style startup idea sharing platform where users can post innovative ideas, explore others’ ideas, and engage through comments and feedback.

The platform is designed to promote **innovation, collaboration, and idea validation** in a clean, scalable, and modern UI experience.

🔗 **Live Site:** https://your-live-site.vercel.app

---

## ✨ Key Features

- 🧠 Share and publish startup ideas with full details (title, description, image, category)
- 💬 Interactive comment system for feedback and discussion
- 🔥 Trending algorithm based on likes and recent engagement
- 🔖 Bookmark system to save favorite ideas
- 👤 Secure authentication system using JWT
- 📊 Personal dashboard to manage user ideas
- 🌓 Fully responsive dark/light mode UI
- ⚡ Optimized performance with Next.js App Router
- 🎨 Modern SaaS-grade UI with smooth animations
- 📱 Mobile-first responsive design
